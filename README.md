## Commands

- Insert mode

| Command | Description |
| -- | -- |
| `jk` | Escape insert |

- Visual mode

| Command | Description |
| -- | -- |
| `cs<target><dest>` | Changes first occurrence of surrounding symbol in line from <target> to <dest> |

- Normal mode

| Command | Description |
| ---- | --- |
| `Ctrl-n` | Toggles neotree sidebar on left |
| `Space-ff` | Toggles telescope fuzzy finder |
